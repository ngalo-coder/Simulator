# Simuatech — Clinical Case Simulator

A React + Express application for medical students to practice clinical diagnosis through interactive AI-powered patient simulations.
## Features

- 🏥 **Patient Cases** — Browse and filter medical simulation cases by category and specialty
- 💬 **AI-Powered Simulation** — Interactive chat with AI patients powered by OpenRouter
- 📊 **Performance Assessment** — Get AI-generated feedback on your diagnosis and interview
